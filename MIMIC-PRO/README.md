Download the folder [here](https://drive.google.com/file/d/1VGfJ8aH9B9GwdNMDPqrYzayk0pN-6e0a/view?usp=sharing) and place it in the `CXR-ReDonE` directory.

Run the following inside `CXR-ReDonE`:

```bash
pip install gdown

unzip data.zip
rm data.zip
```
